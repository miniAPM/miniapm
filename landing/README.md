# MiniAPM Landing Page

Landing page for [miniapm.com](https://miniapm.com).

## Development

Open `index.html` in a browser or use a local server:

```bash
python3 -m http.server 8000
```

## Deployment

Automatically deployed to GitHub Pages on push to `main`.

## Related Repos

- [miniapm](https://github.com/miniapm/miniapm) - Rust server
- [miniapm-ruby](https://github.com/miniapm/miniapm-ruby) - Ruby gem
